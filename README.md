# Balboa-GS510SZ

# PIN

| PIN           | Type            | Description   |
| ------------- | --------------  |-------------- |
| PIN 1         | Power           |  5 VDC        |
| PIN 2         | ?               |               |
| PIN 3         | Digital Input   |  Pump 1       |
| PIN 4         | GND             |  Ground       |
| PIN 5         | Serial Tx       |  Display Data |
| PIN 6         | CLK             |  Clock        |
| PIN 7         | Digital Input   |  Pump 2       |
| PIN 8         | Serial Tx       |  Buttons data |
